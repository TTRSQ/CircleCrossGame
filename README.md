# CircleCrossGame
○✗(マルバツ)ゲーム core for go langage

## 概要
強化学習の勉強の題材に○✗ゲームを選択したが、○✗ゲームから作ることになってしまった。  
人類が○✗ゲームを作るのに割く時間を削減するために作成。
